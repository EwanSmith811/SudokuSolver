# SudokuSolver
Can solve any 9x9 sudoku board, or it will determine if the board is unsolvable
Board must be typed in with a space between each number, an enter to denote a new row, and 0's
as place holders for blanks. Example:
6 0 7 0 0 3 5 9 0
0 0 9 0 0 6 7 0 0
5 0 0 9 0 0 6 0 0
0 5 2 0 0 0 0 1 6
7 0 0 0 2 8 9 0 0
8 0 3 0 0 5 2 0 7
2 0 4 8 0 0 3 0 5
3 0 0 4 5 0 1 8 0
1 0 5 3 0 0 4 7 0

This is a 9x9 board you can paste into the program which it will automatically solve, formatting the answer to appear
as follows:
6 1 7 | 2 4 3 | 5 9 8 
4 2 9 | 5 8 6 | 7 3 1 
5 3 8 | 9 1 7 | 6 2 4 
---------------------
9 5 2 | 7 3 4 | 8 1 6 
7 4 1 | 6 2 8 | 9 5 3 
8 6 3 | 1 9 5 | 2 4 7 
---------------------
2 9 4 | 8 7 1 | 3 6 5 
3 7 6 | 4 5 2 | 1 8 9 
1 8 5 | 3 6 9 | 4 7 2
